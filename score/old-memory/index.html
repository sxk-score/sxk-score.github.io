Old Memory

"Old Memory" is from Yosuga No Sora OST, it is a nostalgic piece that evokes the feeling of reminiscing about the past. It features a gentle melody and a warm, comforting atmosphere.

INTRODUCTION

It's here!
This is a piece from "Yosuga no Sora"!
Although I don't watch "Yosuga no Sora" much, the mood of this piece is definitely there, so I created this.
Different from other versions, this one has a bit of a polished touch to it.
Note: You need at least MuseScore 4.7 to open MSCZ files. If you use an older version of MuseScore or other notation software, then MusicXML is more suitable for you.
Since many people might not have MuseScore 4.7 yet, here is the Beta version download link: Download MuseScore 4.7 Beta
Alternatively, you can download it using the GitHub Actions interface, but this requires a GitHub account.
The reason for using MuseScore 4.7 is because it features ASIO driver support.
